# EfficientAnomalyCamera
Surveillance camera that used EfficientGAN.

```
usage: camera.py [-h] [--device_id DEVICE_ID] [--result_dir RESULT_DIR] [--weight_dir WEIGHT_DIR] [--image_size IMAGE_SIZE] [--lr LR]
                 [--mul_lr_dis MUL_LR_DIS] [--num_scorelist NUM_SCORELIST] [--aug_threshold AUG_THRESHOLD] [--cpu] [--generate GENERATE] [--noresume]

options:
  -h, --help            show this help message and exit
  --device_id DEVICE_ID
  --result_dir RESULT_DIR
  --weight_dir WEIGHT_DIR
  --image_size IMAGE_SIZE
  --lr LR
  --mul_lr_dis MUL_LR_DIS
  --num_scorelist NUM_SCORELIST
  --aug_threshold AUG_THRESHOLD
  --cpu
  --generate GENERATE
  --noresume
```
